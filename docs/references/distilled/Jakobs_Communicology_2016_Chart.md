# Jakobs (2016) — Communicology: Specification of Communication Networks at Four Levels

**Source**: Richard L. Lanigan (chart attributed to Jakobs), "Communicology: Specification of Communication Networks at Four Levels," 2016. Single-page systematic chart extending the Ruesch & Bateson four-level architecture by integrating Jakobson's six communication functions with the R&B network levels.

**Core contribution**: Maps Jakobson's six elements/functions of communication (addresser, context, message, contact, code, addressee) across all four R&B levels (intrapersonal, interpersonal, group, cultural), with critical subdivisions of Level III (egocentric/sociocentric) and Level IV (space-binding/time-binding). Provides the most complete cross-tabulation of communicological structure available in a single schema.

---

## Architecture of the Chart

The chart is organized as a matrix:

**Columns** = Jakobson's six communication elements/functions, each with verbal and nonverbal channel specifications:

| Element | Verbal Channel | Nonverbal Channel |
|---|---|---|
| ADDRESSER | Emotive | Vocalics |
| CONTEXT | Referential | Proxemics & Ocularics |
| MESSAGE | Poetic | Chronemics |
| CONTACT | Phatic | Haptics |
| CODE | Metalinguistic | Kinesics |
| ADDRESSEE | Conative | Olfactorics |

**Rows** = Four media network levels, each functioning as a *medium* of communication.

---

## Level 1: Intrapersonal

| Element | Specification |
|---|---|
| Addresser | **Embodiment**: Self-awareness, synesthesia |
| Context | **Content ordering**: Consciousness |
| Message | **Store signification** |
| Contact | "Within One": Pre-consciousness, pre-reflectivity |
| Code | **Synesthetic meaning** |
| Addressee | **Memory**: Synchronic |

Key features: The intrapersonal is the domain of embodied self-awareness. Communication is *within one* organism. Messages are stored (not transmitted). Meaning is synesthetic (cross-modal). Temporality is synchronic (simultaneous, not sequential). The addresser-addressee circuit is closed within the self. Pre-reflective and pre-conscious contact — one does not "meet" oneself reflectively at this level.

---

## Level 2: Interpersonal

| Element | Specification |
|---|---|
| Addresser | **Dyad**: Self as Other; Relationship |
| Context | **Task ordering**: Experience |
| Message | **Transmit meaning** |
| Contact | "One to One": Consciousness, reflectivity, conscience |
| Code | **Cognitive meaning** |
| Addressee | **History**: Diachronic |

Key features: The interpersonal introduces the dyad — self experienced *as* Other (the fundamental reflexive turn). Communication is one-to-one. Messages are *transmitted* (not merely stored). The contact mode becomes conscious, reflective, and introduces *conscience* (the ethical dimension). Meaning becomes cognitive (not synesthetic). Temporality shifts to diachronic (sequential, historical). Context is ordered by task (not content). The addressee is history itself — interpersonal exchange creates diachronic record.

---

## Level 3: Group

| Element | Specification |
|---|---|
| Addresser | **Community**: Other as Self; Affinity; Kinship |
| Context | **Group ordering**: Socialization |
| Message | **Retrieve signification** |
| Contact | **Present**: Reflexivity; Altruism |
| Code | **Affective meaning** |
| Addressee | **Consociates**: Share Space-Time |

Key features: At the group level, the fundamental reversal occurs — not self-as-other (Level 2) but *other-as-self* (identification, affinity, kinship). Messages are *retrieved* (drawn from collective store, not individually transmitted). The contact mode introduces reflexivity (self-awareness of the group process) and altruism. Meaning becomes affective (neither synesthetic nor cognitive but felt-relational). The addressee is consociates — those who share both space and time (co-present).

### Subdivision: Egocentric vs. Sociocentric

Level III divides into two fundamentally opposed orientations:

**3a. Egocentric (Task Group)**:
- Addresser: Task group — identity by *rule*
- Context: **Aggregate parts** (the whole decomposed into separable units)
- Message: **Centrifugal** — "one to many"
- Contact: **Competition** — creates *agony*
- Code: Names create **static categories**
- Addressee: **Primary roles** — leadership

**3b. Sociocentric (Affiliation Group)**:
- Addresser: Affiliation group — identity by *role*
- Context: **Organic whole** (the parts as inseparable from the whole)
- Message: **Centripetal** — "many to one"
- Contact: **Cooperation** — creates *harmony*
- Code: Names create **dynamic relations**
- Addressee: **Secondary roles** — membership

The egocentric-sociocentric split is structurally parallel to the molar-molecular distinction (D&G) and the filiation-alliance distinction. Egocentric = centrifugal, competitive, categorical, rule-based, aggregate. Sociocentric = centripetal, cooperative, relational, role-based, organic. The egocentric creates agony (struggle, contest); the sociocentric creates harmony. Naming functions differ fundamentally: static categories (fixing identity) vs. dynamic relations (enabling transformation).

## Level 4: Culture

| Element | Specification |
|---|---|
| Addresser | **Co-figurative**: "Peers learn from peers" |
| Context | **Intergroup ordering**: Culturalization |
| Message | **Evaluate, interpret meaning** |
| Contact | **Present**: Reversibility of choice |
| Code | **Conative meaning** |
| Addressee | **Contemporaries**: Share time only |

Key features: At the cultural level, learning becomes co-figurative (peers learning from peers — neither children-from-elders nor elders-from-children, but lateral). The context is intergroup (not intragroup). Messages are *evaluated and interpreted* (not stored, transmitted, or retrieved — the highest communicative operation). The contact mode introduces *reversibility of choice* — at the cultural level, decisions can be revisited, traditions reinterpreted. Meaning becomes conative (directed toward action, volition, will). The addressee is contemporaries — those who share time but not necessarily space (unlike consociates who share both).

### Subdivision: Space-Binding vs. Time-Binding

Level IV divides along the Korzybskian dimension of space-binding vs. time-binding:

**4a. Space/Place (Post-figurative)**:
- Addresser: **Post-figurative** — "Children learn from forebears"
- Context: **Place community** ("Family")
- Message: **Space binding** — Decode: O of E (Object of Expression)
- Contact: "Many to many" ("House") — **Past practice orientation**
- Code: **Digital logic** — In-group vs. out-group
- Addressee: **Predecessors**: Previous generations; ancestors

**4b. Time/Moment (Pre-figurative)**:
- Addresser: **Pre-figurative** — "Adults learn from children"
- Context: **Non-place community** ("Profession")
- Message: **Time binding** — Incode: O of A (Object of Action)
- Contact: "Many to many" ("Home") — **Future practice orientation**
- Code: **Analogue logic** — Diffusion of innovations
- Addressee: **Successors**: Succeeding generation; progeny

The space/time subdivision captures the deepest cultural distinction. Space-binding = post-figurative (tradition flows backward from ancestors), place-based ("Family" as spatial community, "House" as physical structure), past-oriented, digital/binary (in-group/out-group), decoding (extracting Object from Expression). Time-binding = pre-figurative (novelty flows forward from the young), non-place ("Profession" as temporal community, "Home" as existential dwelling), future-oriented, analogue/continuous (diffusion of innovations), incoding (inscribing Object into Action).

The "House" vs. "Home" distinction is particularly significant: House = physical structure (place); Home = existential dwelling (time). Both involve "many to many" communication but with opposite temporal orientations.

## Structural Principles

1. **Ascending complexity of message operations**: Store (L1) → Transmit (L2) → Retrieve (L3) → Evaluate/Interpret (L4). Each level adds a higher-order operation on messages.

2. **Meaning type rotation**: Synesthetic (L1) → Cognitive (L2) → Affective (L3) → Conative (L4). This sequence follows Jakobson's functional hierarchy but maps it onto communicological levels.

3. **Temporal progression**: Synchronic (L1) → Diachronic (L2) → Space-Time shared (L3) → Time-only shared (L4). Progressive detachment from spatial co-presence.

4. **Contact mode progression**: Pre-reflective (L1) → Reflective/Conscience (L2) → Reflexive/Altruistic (L3) → Reversible choice (L4).

5. **Margaret Mead's three figurative modes** distributed across levels: Post-figurative (4a, tradition), Co-figurative (4 base, peer learning), Pre-figurative (4b, innovation from the young).

6. **Korzybski's binding dimensions**: Space-binding (4a, rooted, territorial) vs. Time-binding (4b, mobile, professional). This maps directly onto Emery & Trist's Type II (positional, place-based) vs. Type III/IV (operational, capacity/power-based) environmental types.

## Key Concepts for sigma-TAP Integration

1. **Four-level message operations map to L-matrix**: Store (L11, intrapersonal) → Transmit (L12, org→env) → Retrieve (L21, env→org) → Evaluate/Interpret (L22, env↔env). The ascending complexity of operations directly mirrors the L-matrix quadrants.

2. **Egocentric/sociocentric split = molar/molecular**: The Level III subdivision maps onto the arborescent/rhizomatic, filiation/alliance, competitive/cooperative distinctions. Egocentric = centrifugal, categorical, leadership-structured. Sociocentric = centripetal, relational, membership-structured.

3. **Space-binding/time-binding = territorial/deterritorialized**: Space-binding (place community, past orientation, digital in-group/out-group) = territory. Time-binding (non-place community, future orientation, analogue diffusion) = deterritorialization.

4. **Meaning type rotation maps to TAPS**: Synesthetic (T — embodied becoming), Cognitive (A — being-state), Affective (P — relational acting), Conative (S — volitional creating). The rotation through meaning types corresponds to the TAPS signature cycle.

5. **House vs. Home = facade vs. face**: House (spatial structure, place community) corresponds to facade (exhibited surface). Home (existential dwelling, non-place community) corresponds to face (capacity to exceed surface).

6. **Co-figurative base level + post/pre subdivisions**: The cultural level's base is peer-to-peer (co-figurative), subdivided into backward-looking (post-figurative, ancestors) and forward-looking (pre-figurative, successors). This triadic structure parallels the hexis-praxis-metathesis triad.

7. **Decode (O of E) vs. Incode (O of A)**: Space-binding decodes — extracts Object from Expression (reading the world). Time-binding incodes — inscribes Object into Action (writing the world). This maps to consumption (process-praxis, taking up sedimented structure) vs. consummation (praxis-process, praxis reaching its term).

8. **Agony vs. Harmony**: The egocentric group creates agony (competitive struggle), the sociocentric creates harmony (cooperative resonance). This is not a value judgment but a structural description of two irreducible group orientations. Maps to the adversity-coefficient vs. prosperity-coefficient.
