## Integration Points

- **Tetravalent assemblage = L-matrix**: This is the most structurally important mapping. D&G's tetravalent assemblage (content × expression, territory × deterritorialization) provides the fourth independent convergence on 4-channel structure, alongside R&B (communication levels), Emery (causal textures), and Lizier (information channels). The convergence web should link this to all three.
  - **Candidate forward notes**: The content/expression axis may provide an orthogonal classification WITHIN each L-channel — i.e., every L12 event has both a content aspect (what action was performed) and an expression aspect (what sign was produced). This is NOT currently captured in the L-matrix ledger.
  - **Concept map entries**: `DG:TetravalentAssemblage` maps to L-matrix four-channel structure (confirms w:78-81 RB, w:82-83 Emery).

- **Four forms of D as system trajectory taxonomy**: The four D forms provide a richer state-space for system dynamics than the binary beneficial/detrimental:
  - Negative D → agent/system locked in overconservation (high L11, low L12)
  - Relative positive D → agent innovating but captured (L12 active but segmented by L22)
  - Absolute creative D → paradigm shift (all four L-channels reconfigure = four time-modes in phase)
  - Absolute destructive D → catastrophic collapse (demetathesis cascade without syntegrative recovery)
  - **Candidate forward notes**: §5.26 beneficial/detrimental apocalypses should be expanded to four forms per D&G. The two "absolute" forms (creative vs destructive) correspond to the existing beneficial/detrimental, but the two "relative" forms (negative, positive) are novel — they describe sub-apocalyptic dynamics currently undifferentiated in sigma-TAP.
  - **Concept map entries**: `DG:FourFormsOfD` maps to apocalypse mechanisms + system trajectory states.

- **Selection rule = conservation law**: D&G's selection rule ("only what increases the number of connections at each level") is structurally identical to the conservation law (§5.43: minimal praxis for maximal syntegration). This is an independent convergence with Fermat/Feynman (w:76-77) — D&G arrive at a variational principle from continental philosophy rather than physics.
  - **Concept map entries**: `DG:SelectionRule` maps to conservation law §5.43. Convergence web entry with Fermat/Feynman:least_action.

- **Conjunction vs connection → junction typology**: D&G explicitly oppose conjunction (blocking/gating) to connection (creative multiplication). This directly grounds the project's junction/conjunction/disjunction framework and the connection fallacy (§5.61).
  - **Concept map entries**: `DG:ConjunctionVsConnection` maps to junction typology §5.61.

- **Mutual immanence → hexis-praxis grounding**: The molar/molecular mutual immanence provides the ontological ground for the hexis-praxis cycle. It's not alternation between two opposed states but mutual immanence — each issues from the other. This reframes §5.19 from a temporal cycle to an ontological co-implication.
  - **Candidate forward notes**: §5.19 hexis-praxis description should be updated to reflect mutual immanence (D&G) rather than simple oscillation.

- **Quantitative analysis legitimation**: D&G's explicit insistence on "quantitative analysis" and "coefficients" for assemblages is direct philosophical authorization for sigma-TAP's computational approach. The simulation is not reducing philosophy to computation but following D&G's own prescription for schizoanalysis.

## Open Questions

1. **Content/expression as orthogonal to L-channels**: If every L-channel event has both a content (pragmatic) and expression (semiotic) aspect, does this suggest a 4×2 = 8-fold classification of agent interactions? Currently the L-matrix records WHICH channel, but not the content/expression character of the event within that channel. Is this a missing dimension?

2. **Phylum vs diagram**: D&G distinguish matter-movement (phylum) from function-expression (diagram) within the abstract machine. Does this map to the dM_history (trajectory = phylum) vs L-matrix (distribution = diagram) split? If so, the TAPS signature already implicitly captures both — but are they being tracked as genuinely distinct?

3. **Three types of abstract machines**: The mapping to three sigma-TAP subsystems (cross-metathesis = consistency, TAPS classification = stratification, L22 regulation = axiomatic) is suggestive but needs testing. Are there assemblages in the simulation that exhibit all three machine types simultaneously? D&G insist all three "overlap and intertwine."

4. **"Concrete rules of extreme caution"**: This appears in the chapter TITLE — it's not a side observation but the central prescription. How do the "concrete rules" for controlled destratification translate into parameter constraints in sigma-TAP's apocalypse mechanisms? Is there a formal rule for "too-sudden" destratification (a rate threshold)?
