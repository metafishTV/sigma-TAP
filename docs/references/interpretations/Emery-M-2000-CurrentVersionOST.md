## Integration Points

- **DirectiveCorrelation=sigma(Xi)**: Sommerhoff's DC provides the formal definition of what σ(Xi) measures — degree of correspondence between agent trajectory and environmental field, correlated in terms of direction. σ high = directive correlation (adaptive); σ → 0 = no directive correlation (maladaptive). The temporal dimension (t0 → t1 → t2) means σ should capture trajectory direction, not just snapshot correspondence. This is the difference between purposive adaptation (tracking current L22) and ideal-seeking (tracking potential future L22 states).

- **PotentialDC=TAPS-T**: Potential directive correlation — the capacity to imagine and expect — maps to the TAPS T-component (how I become). This is the agent's capacity to envision states not yet realized and act toward them. In simulation: agents with dominant T would exhibit ideal-seeking behavior (pursuing novel type combinations rather than replicating existing ones). This extends the TAPS framework by grounding T in Sommerhoff's formal mechanism.

- **TransactionOntology=CrossMetathesis**: Every cross-metathesis event must be transactional — BOTH participants change. Absorptive: receiver absorbs types, giver's L12 count increments and TAPS shifts. Novel: both parents become dormant while child spawns. The L-matrix already tracks quantitative change; the experiential change (the giving agent becoming different through having given) should be reflected in agent state, not just counters.

- **UnitasMultiplex=FieldArchitecture**: Angyal's system principle validates the field architecture over pairwise interaction models. Group L11 is not the sum of pairwise L12/L21 exchanges but the emergent organizing principle. Two groups with identical pairwise exchange patterns but different L11 principles (different TAPS signatures, different design principles) will behave differently. This is already implicit in the per-agent L-matrix ledger but should inform group-level analysis.

- **DP1DP2=StructuralAttractors**: DP1 and DP2 as testable simulation predictions: DP2 conditions (self-managing coordination, shared responsibility) → continuous type_set growth, rising TAPS diversity, temporal state progression. DP1 conditions (hierarchical coordination, imposed goals) → type_set atrophy, signature homogenization, temporal stagnation. Measure type_set diversity and temporal trajectories under each condition.

- **EnvironmentalType=EmergentMeasurement**: Type II/III/IV as emergent simulation properties, not parameters. A simulation producing stable DP2 groups in cooperative social field = Type II. DP1 hierarchies competing for resources = Type III. Environmental dynamics overwhelming all organizational responses = Type IV. Environmental type should be measured from simulation output.

- **FourIdeals=L-channel-TAPS**: Speculative mapping: Homonomy (L21) ↔ A (how I am); Nurturance (L12) ↔ P (how I act); Humanity (L11) ↔ T (how I become); Beauty (L22) ↔ S (how I create). If validated, each TAPS letter would correspond to an L-channel, a temporal mode, AND a fundamental human ideal. Agent TAPS profile = ideal-orientation profile.

- **PuzzleLearning=NovelCrossMetathesis**: Puzzle learning (unknown endpoint) maps to novel cross-metathesis — the emergent type has never existed, discovered through the combination process. Problem solving (known endpoint) maps to absorptive cross-metathesis — the target type_set configuration is known. Simulation prediction: stable environments favor absorptive (problem solving); turbulent environments demand novel (puzzle learning).

- **TwoStageSequence=ArchitecturePhasing**: SC → PDW sequence maps to planned Stage 3C+ architecture: Phase 1 (sigma feedback + trust = SC equivalent) changes L11 ↔ L22 relation. Phase 2 (topology + family groups = PDW equivalent) changes intra-L11. Sequence is mandatory — agents that restructure without environmental feedback produce internally coherent but maladapted organizations.

- **EmbryosOfChange=MetatheticIndicators**: Embryos of social change map to early metathetic events as paradigm shift indicators. Diagnostic: novel cross-metathesis events clustered in a few agents = embryos forming (emergent paradigm shift). Distributed randomly = background noise. Clustered novelty is the signal for detecting value shifts before they become system-wide.

- **FunctionalLevelHierarchy=AgentCapacityLevels**: Three levels (goal-seeking → purposefulness → ideal-seeking) could map to agent maturity or temporal state. Agents restricted to goal-seeking (DP1 conditions) cannot access ideal-seeking. Only agents in DP2 conditions have the extra degree of freedom. This creates a testable asymmetry between design principle conditions.

- **StructuralCorroboration=ValidationMethod**: The paper prescribes structural corroboration (same genotype across different contexts) as the validation method for OST(E). For sigma-TAP: the test is not replication under identical conditions but same structural invariants (DP1/DP2 attractors, environmental type dynamics, six criteria effects) appearing under DIFFERENT parameter settings, initial conditions, and configurations. This is the strongest form of model validation.

## Open Questions

1. **Ideal → L-channel → TAPS mapping**: The four ideals (Homonomy, Nurturance, Humanity, Beauty) → L-channels is structurally implied but never stated explicitly by Emery. The further extension to TAPS letters (A, P, T, S) is speculative. Should this triple mapping be treated as confirmed or as a hypothesis requiring independent validation?

2. **DP1/DP2 as continuous vs binary**: Emery presents DP1/DP2 as a binary genotypical distinction — an organization is structured on one or the other. In sigma-TAP, could there be a continuous spectrum (degree of DP1 vs DP2) or intermediate states? The "laissez-faire" question from deGuerre also applies here — is there a structural zero-point where neither principle operates?

3. **Ecological learning and agent perception**: Emery (via Gibson) claims people directly extract meaningful information from environments without internal model mediation. How does this map to sigma-TAP agents who currently use σ feedback as a mediated signal? Does the Gibson claim require a more direct agent-environment coupling than the current sigma feedback mechanism provides?

4. **Coimplication formalization**: Figure 3 uses L11(L22) and L22(L11) notation to express mutual determination. This is stronger than bidirectional causation — each IS a function of the other. How should this coimplication be formalized in the simulator? Currently L11 and L22 are tracked independently with L12/L21 as exchange channels. Does coimplication require a structural coupling beyond exchange?
